Full-stack development requires proficiency in both front-end and back-end technologies, allowing developers to build and maintain entire web applications. Front-end skills include knowledge of HTML, CSS, JavaScript, and modern frameworks like Angular, React, or Vue.js, enabling the creation of interactive and user-friendly interfaces. Huehiw Marketplace

React, TS, CSS, HTML, JS

Huehiw Marketplace is an online marketplace application developed using CSS, JavaScript, ReactJS, and TypeScript. The application offers key features that enable users to conveniently list items for sale, search for products, and manage their profiles. Its standout functionalities include an intuitive post creation interface, an explore page for browsing posts, profile editing, and the ability to like or save posts. Users can also edit and delete their own content as needed. The platform incorporates monetization features such as banner advertisements, post boosting services to increase visibility, and a membership system designed to enhance promotional opportunities. This application is designed to be user-friendly while empowering users to maximize their market potential

Linebot Spam Detector

HTML, CSS, Python, Flask, JS

The Thai Language Spam Detector with a Line Bot and Website is a comprehensive project that combines artificial intelligence, web development, and chatbot technology to tackle spam message detection in the Thai language. The project utilizes a custom-trained AI model based on LSTM (Long Short-Term Memory) to classify text as spam or non-spam effectively. The training dataset was meticulously crafted by collecting spam messages, curating non-spam examples, and generating additional data using Python-based scripts.
